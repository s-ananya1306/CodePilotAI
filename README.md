# 🚀 CodePilotAI

CodePilotAI is an AI-powered coding assistant that helps developers understand, search, and analyze source code repositories using Retrieval-Augmented Generation (RAG).

## Features

- 📂 Upload and extract Git repositories
- 🔍 Repository scanning and code parsing
- ✂️ Intelligent code chunking
- 🧠 Embedding generation using Sentence Transformers
- 📚 FAISS Vector Database
- 🤖 Semantic code search
- 🧩 Multi-Agent AI workflow (Coming Soon)
- 💬 LLM-powered code explanations (Coming Soon)
